Skills Table

Skill Area Category Level What to Understand (Deep) Suggested Hands-on Practice
HTML - BasicsHTMLBeginnerElements, tags, attributes, document structure (doctype, head, body).Create simple pages, forms, lists, links.
Semantic HTMLHTMLIntermediateUse semantic tags (header, nav, main, article, section, footer) for meaning & SEO.Refactor a page to replace divs with semantic tags; improve SEO.
Forms & ValidationHTMLIntermediateForm controls, input types, validation attributes, accessibility concerns.Build multi-field forms with validation and accessible labels.
HTML APIsHTMLAdvancedUnderstanding of DOM, history API, templates, data-* attributes.Use templates and manipulate DOM with vanilla JS for progressive enhancement.
Accessibility (a11y) - HTMLHTMLAdvancedARIA roles, semantic landmarks, focus management and keyboard interaction.Audit pages with screen-reader testing and fix issues.
CSS - BasicsCSSBeginnerSelectors, box model, margin/padding, fonts, colors, specificity basics.Style a static page with typography and spacing.
Layout - Normal Flow & PositioningCSSIntermediateDisplay types, position (static/relative/absolute/fixed/sticky), floats, z-index.Convert a mockup to use positioning and stacking context.
FlexboxCSSIntermediateOne-dimensional layout for rows/columns, alignment, order, flex properties.Build navbars, card rows, vertical centering with flexbox.
GridCSSAdvancedTwo-dimensional layout: grid-template, areas, fractions, auto-placement.Create complex responsive page layouts with CSS Grid.
Responsive DesignCSSAdvancedMedia queries, fluid typography, container queries (where supported), mobile-first.Make a multi-breakpoint responsive site from a design.
Modern CSS (Custom Properties)CSSAdvancedCSS variables, calc(), clamp(), modern functions for dynamic sizing.Implement theme variables and responsive sizing with clamp.
Preprocessors & ToolingCSSIntermediateSASS/SCSS, nesting, variables, mixins, build pipeline basics (PostCSS).Refactor repetitive CSS into SCSS with variables/mixins.
Design Systems & ArchitectureCSSAdvancedBEM, SMACSS, utility-first, componentization, tokens and theming.Create a mini design system and component stylesheet.
Performance & Critical CSSCSSAdvancedMinification, critical-path CSS, render-blocking minimization, file splitting.Analyze and optimize CSS delivery for faster first paint.
Accessibility (a11y) - CSSCSSAdvancedVisible focus states, reduced-motion support, color contrast considerations.Ensure focus visibility and add prefers-reduced-motion support.
Testing & DebuggingToolsIntermediateDevTools, Lighthouse, browser compatibility testing, cross-device checks.Use DevTools to debug layout, measure performance, and fix issues.
Progressive Enhancement & Graceful DegradationPrinciplesAdvancedBuild for baseline functionality then enhance; avoid breaking for older browsers.Provide accessible fallbacks for modern features.
Integration with JS & FrameworksPrinciplesIntermediateHow CSS & HTML interact with frameworks and JS (scoped CSS, CSS-in-JS considerations).Style components in a small framework app and manage styles lifecycle.
Soft Skills & WorkflowNon-technicalBeginnerCollaboration, Git-based workflow, code review, semantic commits.Contribute to a small frontend repo and follow PR process.
